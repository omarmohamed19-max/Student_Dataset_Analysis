# 🎓 Student Dataset Analysis

An end-to-end Python data analysis project demonstrating data exploration, cleaning, analytical queries, vectorization using NumPy, and data visualisations.

---

## 📌 Project Overview

This project analyzes a dataset of students' performance and attendance across various departments. The workflow is divided into 5 structured phases:

1. **Exploratory Data Analysis (EDA):** Initial inspection of dataset shape, head/tail records, and column types.
2. **Data Cleaning:** Handling missing numerical values (Imputation), eliminating duplicate records and duplicate Student IDs.
3. **Data Analysis:** Creating metrics (`Total`, `Average`, `Grade`), department/gender distribution analysis, and performance filtering.
4. **NumPy Array Manipulations:** 2D array transformations, matrix indexing/slicing, array arithmetic (bonus mark additions), and value capping with `np.clip()`.
5. **Data Visualization:**
   - **Bar Chart:** Average performance per subject.
   - **Pie Chart:** Percentage distribution across academic departments.
   - **Histogram:** Spread & distribution of student overall averages.
   - **Line Chart:** Performance tracking of the Top 10 students.

---

## 🛠️ Tech Stack & Libraries Used

- **Python 3**
- **Pandas:** Data loading, cleaning, manipulation, and analysis.
- **NumPy:** Vectorized array operations and numerical boundaries.
- **Matplotlib:** Data visualizations and custom plot formatting.
- **OpenPyXL:** Excel file I/O handling.
